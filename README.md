## Overview

Responsive QR code component inspired by the Frontend Mentor design. It centers the card on a soft slate background, displays the supplied QR image, and matches the typography/spacing guidance from the style guide.

## Design notes

- Mobile-first card at `min(360px, 100%)` width with a subtle shadow to match the mock.
- Color palette uses the supplied slate tones and white background from `style-guide.md`.
- Typography is Outfit with text sizes that mirror the design (1.75rem heading, ~0.94rem body).
- Footer links keep the Frontend Mentor attribution while allowing you to swap in your personal handles.

### What I learned

- How to recreate a clean component by translating spacing hints from a JPG design.
- Using `min()`/`max-width` to keep the card readable across narrow and wide viewports.
- Crafting a minimalist style sheet with a single embedded `<style>` block for quick prototyping.
